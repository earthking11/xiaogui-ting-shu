# 小龟听书

小龟听书是一款 Android TXT 小说阅读和听书 App。

你可以把本地 TXT 小说导入 App 阅读；读累了，也可以用 MiMo TTS 把当前内容朗读出来。

## 主要功能

- 导入本地 `.txt` 小说文件。
- 支持从文件管理器或其他 App 分享 TXT 到小龟听书。
- 自动记录每本书的阅读进度。
- 支持书签、主题、字号和行距设置。
- 右下角显示当前阅读百分比。
- 可填写 MiMo API Key，把小说内容转换成语音朗读。
- 提供 API 连通性测试按钮，能显示更具体的失败原因。
- 听书时显示播放进度、当前朗读片段和下一段生成状态。

## 如何使用

### 导入小说

1. 打开小龟听书。
2. 点击 `导入 TXT`。
3. 选择手机里的 TXT 小说文件。
4. 导入后进入阅读页。

你也可以在文件管理器里选中 TXT 文件，点击系统 `分享`，再选择 `小龟听书`。

### 阅读

- 上下滑动阅读正文。
- 点击屏幕空白处，可以显示或隐藏顶部信息栏和底部工具栏。
- 点击 `阅读及 TTS设置` 可以调整主题、字号、行距、音色和语速。
- 点击书签按钮可以保存当前阅读位置。

### 听书

1. 在阅读页点击 `我想听书`。
2. 首次使用时，按提示填写 MiMo API Key。
3. 也可以先进入 `阅读及 TTS设置`，填写 Key 后点击 `测试 API 连通性`。
4. 连通性正常后，再点击 `我想听书` 开始朗读。

听书开始后，底部会出现一个小播放器，显示当前播放进度和下一段是否已经生成。

## 关于 MiMo API Key

小龟听书不会内置任何 API Key。你需要自己准备 MiMo API Key，并在 App 内填写。

Key 只保存在你的手机本机，用于向 MiMo 发起 TTS 请求。项目源码、README 和 APK 中都不包含真实 Key。

如果朗读失败，可以先用设置页里的 `测试 API 连通性` 判断问题原因，例如：

- API Key 无效
- 账户余额不足
- 请求过于频繁
- 网络不可用
- 服务暂时不可用
- 连接超时或中断

## 隐私和权限

- TXT 文件会复制到 App 私有目录中保存。
- App 不申请全盘文件管理权限。
- App 不会把你的 MiMo API Key 写进源码或安装包。
- 朗读时，当前待朗读文字会发送给 MiMo TTS 服务生成音频。

## 当前限制

- 当前版本主要面向 Android 手机。
- 安装包是 debug 测试版，不是应用商店正式签名版。
- 目前听书是分段生成音频，不是实时流式朗读。
- 重新打开 App 会恢复阅读位置，但不会自动恢复到上次音频播放秒数。
- 后台播放、锁屏控制、章节目录识别等功能还在后续计划中。

## 适合谁

如果你有 TXT 小说文件，想在手机上安静阅读，并在需要时用 TTS 听书，小龟听书就是为这个场景做的。
