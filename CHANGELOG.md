# 更新日志

## v1.1 - 2026-06-12

这是一次体验打磨版本，重点优化阅读页、朗读设置、错误反馈和进度恢复。

### 阅读体验

- 底部工具栏中间入口从 `Aa` 改为 `阅读及 TTS设置`，入口含义更清楚。
- 阅读页底部留白会根据工具栏和听书播放器自动调整，减少正文被遮挡。
- 右下角阅读百分比在工具栏或听书播放器出现时会自动上移。
- 强杀 App 后重新进入，会按实际可见位置重新计算阅读百分比，避免短书场景下百分比和页面位置不一致。
- 书签提示更明确，会显示当前已读百分比；重复书签提示改得更自然。
- 阅读页提示条会根据底部播放器和工具栏上移，不再压住底部操作区。

### 阅读及 TTS 设置

- 设置页标题改为 `阅读及 TTS 设置`。
- 设置页拆分为 `阅读外观`、`朗读设置`、`MiMo API Key`，结构更清楚。
- 主题、字号、行距、音色、语速现在点击后会立刻刷新选中态，不再像“点了没反应”。
- `保存 Key` 在输入为空时会立刻显示提示。
- 保存、清除和连通性测试结果改为页内反馈卡片，并用颜色区分成功和失败。
- API Key 输入框继续使用普通文本输入，避免触发部分安卓系统的安全键盘。

### 听书与错误反馈

- 首次听书弹出的 API Key 输入框也改为普通文本输入，不再触发安全键盘。
- 听书播放器按钮文案改为更短的 `暂停`、`继续`、`重试`、`填 Key`，小屏显示更稳。
- 假 Key 或认证失败时，播放器会提示 `请检查 Key 或网络后重试`，不再误显示 `正在生成快速起播段`。
- 更多面板中的朗读状态改为中文，例如 `正在朗读`、`准备下一段`、`出错`，不再显示开发者状态名。
- 更多面板中的 Key 入口改为 `朗读与 Key 设置`，并只显示掩码后的 Key。

### 验证

- 已在 Android 模拟器中验证 TXT 导入、阅读、设置即时反馈、空 Key 提示、假 Key 连通性错误、假 Key 听书错误、书签提示、强杀恢复进度。
- `flutter analyze` 通过。
- TTS 相关测试通过。
- Android arm64 debug APK 构建通过。

## v1.0 - 2026-06-12

首个可用测试版本。

- 支持导入本地 TXT 小说。
- 支持从系统文件选择器读取 TXT。
- 支持阅读进度恢复、主题、字号、行距、书签。
- 支持填写 MiMo API Key。
- 支持 MiMo TTS 分段生成与播放。
- 支持快速起播段和下一段预生成。
- 支持底部听书播放器和右下角阅读百分比。
